## 8. UI Components

### CommandMenu (`src/components/ui/CommandMenu.tsx`)
- Opens with `Cmd+K` / `Ctrl+K`
- Commands organized in 3 categories:
  - **Navigation:** Go Home, About Ganesh, View Projects, Contact Me
  - **Actions:** Download Resume, Copy Email
  - **Preferences:** Toggle Theme
- Keyboard: Arrow Up/Down to navigate, Enter to execute, Escape to close
- Lazy-loaded (SSR: false) via dynamic import in ThemeProvider

### CustomCursor (`src/components/ui/CustomCursor.tsx`)
- Two elements: small accent dot (follows instantly) + larger ring (lerps with 0.15 factor)
- Expands/fills on hover over interactive elements (a, button, input, etc.)
- Hidden on touch devices via CSS (`@media (hover: none)`)
- Lazy-loaded (SSR: false) via dynamic import in ThemeProvider
- `body { cursor: none }` set in globals.css

### MagneticButton (`src/components/ui/MagneticButton.tsx`)
- Wraps any children; on mousemove, translates toward cursor by `strength * offset` (default: 0.4)
- Resets to (0,0) on mouseleave
- Used for: Email Me, View Projects, Resume, Copy email, Send Email buttons

### ScrollProgress (`src/components/ui/ScrollProgress.tsx`)
- Fixed top-0 bar, 2px height, z-index 99997
- Gradient: `#8B5CF6` → `#EC4899` (violet to pink)
- Uses pure DOM + rAF — no Framer Motion — for perf
- `scaleX` transform for GPU-composited animation
- Rendered in root layout AND again in CaseStudyClient (double instance on case study pages — may stack)

### SectionWrapper (`src/components/ui/SectionWrapper.tsx`)
- Simple wrapper; not currently used in main sections (sections handle their own padding)

## 12. Animation Patterns

### Framer Motion (used in most components)
- **Entrance:** `initial={{ opacity: 0, y: 20/30/40 }}` → `whileInView={{ opacity: 1, y: 0 }}`
- **Stagger:** `containerVariants` with `staggerChildren: 0.1` in Hero
- **Viewport:** `{ once: true, margin: "-60px" }` — triggers once, 60px early
- **Easing:** `[0.25, 0.1, 0.25, 1]` (custom cubic-bezier) for smooth feel
- **Scale effects:** Contact email card scales from `0.96`
- **AnimatePresence:** Used for Navbar mobile menu + CommandMenu modal

### CSS Animations (GPU-optimized, in globals.css)
| Name | Usage | Duration |
|---|---|---|
| `float` | Profile card floating | 6s |
| `spin` | Profile ring gradient | 12s |
| `pulse-ring` | Status dot pulse | 2.5s |
| `scrollBounce` | Scroll indicator | 2s |
| `shimmer` | Skeleton loading | 1.5s |
| `grain` | Film grain texture | 8s steps |
| `blurReveal` | Text reveal | 0.6s |

### Performance Decisions
- `will-change: transform` on animated elements
- No `backdrop-filter` on glass cards (replaced with semi-transparent solid bg)
- Project cards use `box-shadow` changes on hover (not transform) to avoid layout
- Custom cursor and scroll progress use pure DOM + `requestAnimationFrame`, no React state
- Framer Motion lazy-loaded; `optimizePackageImports` in next.config.ts

## 16. Quick-Change Guide

### To add a new project:
1. Edit `src/data/projects.ts` — add a new object to the `projects` array
2. Follow the `Project` interface exactly
3. The Work section and case study route auto-pick it up
4. Add images to `/public/images/` and update `heroImage`/`mockupImage` paths

### To update personal info:
- **Name/title/bio text:** `About.tsx` (bio paragraphs, stats, experience)
- **Skills/toolbox:** `About.tsx` — `skills` array and toolbox list
- **Hero tagline/description:** `Hero.tsx` — the `<p>` description text
- **Experience timeline:** `About.tsx` — `experiences` array
- **Contact info:** `Contact.tsx` — location, timezone, response time text

### To change colors/accent:
1. `globals.css` → `:root` and `.dark` blocks (CSS variables)
2. `tailwind.config.ts` → `theme.extend.colors`

### To add a new section:
1. Create `src/components/sections/NewSection.tsx`
2. Add `id="new-section"` to the section element
3. Import and add to `src/app/page.tsx`
4. Add to Navbar `navLinks` array in `Navbar.tsx`
5. Add to CommandMenu commands in `CommandMenu.tsx`

### To change fonts:
1. `src/app/layout.tsx` — change the `next/font/google` import and variable
2. `globals.css` — update `font-family` on `body`
3. `tailwind.config.ts` — update `fontFamily` extension

### To enable Lenis smooth scroll:
1. Import `LenisProvider` in `src/app/layout.tsx`
2. Wrap `<body>` content with `<LenisProvider>`

### To add a dedicated case study page (like PhonePe):
1. Create `src/app/work/your-slug/page.tsx`
2. In `src/app/work/[slug]/page.tsx`, add the slug to the filter or redirect
